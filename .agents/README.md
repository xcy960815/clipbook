# Task Guides

Use these files as task-specific addenda to the root `AGENTS.md`.

- `local-build.md`
  Use for local `xcodebuild` runs, app bundle delivery, cleanup, and build artifact discipline.
- `settings-native-ui.md`
  Use for settings-window UI work, shortcut recorders, focus behavior, and AppKit/SwiftUI native consistency.
- `testing.md`
  Use for deciding test scope, running unit/UI tests, and documenting manual verification or local test limitations.
- `release-and-sparkle.md`
  Use for version bumps, commits, tags, release preparation, and Sparkle signing/update-chain work.

Load the smallest relevant guide set for the current task.
